#  MetaBox Plugin (WordPress)

A custom **MetaBox Plugin for WordPress** that allows developers and site administrators to easily add, manage, and display custom meta boxes on posts, pages, and other post types.

This plugin demonstrates how to extend WordPress using custom fields and meta boxes, giving you control over additional content without relying on third-party plugins.

---

##  Key Features

- Custom meta box creation
- Simple and clean UI in the WordPress admin
- Easy customization and extendability
- Works with posts, pages, and custom post types
- Lightweight and optimized for performance
- Compatible with modern WordPress environments

---

##  Tech Stack

| Technology | Purpose |
|------------|---------|
| PHP        | Main plugin logic |
| WordPress API | Plugin integration |
| HTML       | Render meta boxes |
